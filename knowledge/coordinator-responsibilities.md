# Coordinator Responsibilities

1. Select execution mode.
2. Enforce Discover before Plan.
3. Ensure plan approval before Work.
4. Run Critical findings as hard gates.
5. Always complete [[phases/compound]].
6. Track cycle metrics.
7. Consult knowledge files every cycle.
8. Escalate and handle `SCOPE_ELEVATION` when scope changes.
