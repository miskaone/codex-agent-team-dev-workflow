# Example Project: Todo CLI (using agent-team-dev-workflow)

Minimal working example showing the full **Codex Agent Team Development Workflow** in action.

## Quick Start
1. Copy this folder into your Codex workspace
2. Run the workflow: `build` / `fix` / `implement priority support`
3. All artifacts land in `.workflow/current/`
4. Learnings are automatically compounded

**Tech stack**: Pure Python 3.11+ (stdlib only)
